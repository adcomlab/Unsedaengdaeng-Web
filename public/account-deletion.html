무료운세댕댕 회원 탈퇴 절차

앱 내에서 직접 삭제 (권장)

가장 간편한 방법은 앱 내에서 직접 삭제하는 것입니다.

1. 무료운세댕댕 앱 실행
2. 하단 탭바에서 마이페이지 진입
3. 화면 하단의 회원 탈퇴 버튼 선택
4. 안내 사항 확인 후 탈퇴 확인

이메일로 요청

앱에 접근할 수 없는 경우 이메일로 삭제 요청 가능합니다.

  • 수신처: support@unsedd.com
  • 제목: [계정 삭제 요청] 가입 이메일
  • 본문: 가입 시 사용한 이메일과 닉네임
  • 처리 기간: 영업일 기준 3일 이내

삭제되는 데이터

  • 이메일, 닉네임, 프로필 사진, 소셜 로그인 정보
  • 사주/궁합 프로필, AI 해설 결과
  • 운세 기록, 발송/수신 선물
  • 푸시 알림 토큰, 앱 사용 로그

보관되는 데이터 (법령 준수)

관련 법령에 따라 다음 데이터는 일정 기간 보관됩니다.

  • 결제·환불 기록: 5년 (전자상거래법)
  • 전자금융 거래 기록: 5년 (전자금융거래법)
  • 접속 로그: 3개월 (통신비밀보호법)

이 정보는 삭제되지 않지만 식별자가 익명화되어 개인을 특정할 수 없습니다.

주의: 회원 탈퇴 시 보유한 간식·발바닥은 환불되지 않으며, 삭제된 데이터는 복구할 수 없습니다.

문의

계정 삭제와 관련하여 문의가 있으시면 support@unsedd.com으로 연락해주세요.
