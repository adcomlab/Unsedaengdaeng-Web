전체 데이터 삭제

모든 개인 데이터를 한 번에 삭제하려면 회원 탈퇴를 진행하세요.

👉 자세한 절차: 계정 삭제 안내

일부 데이터 삭제

특정 항목만 삭제를 원하실 경우 앱 내에서 다음과 같이 처리할 수 있습니다.

  • 사주 프로필 삭제: 사주 목록 → 항목 우상단 점 3개 메뉴 → 삭제
  • 궁합 결과 삭제: 궁합 목록 → 항목 길게 누르기 → 삭제
  • 푸시 알림 토큰 해제: 마이페이지 → 알림 설정 → 끄기
  • 마케팅 수신 동의 철회: 마이페이지 → 설정 → 마케팅 수신

이메일로 데이터 삭제 요청

앱에서 직접 삭제할 수 없는 경우 이메일로 요청 가능합니다.

  • 수신처: support@unsedd.com
  • 제목: [데이터 삭제 요청] 가입 이메일
  • 본문: 가입 이메일, 삭제를 원하는 데이터 항목
  • 처리 기간: 영업일 기준 3일 이내
